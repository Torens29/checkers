# Шашки/checkers

**Классическая игра «Шашки» для двух игроков**  
Логическая настольная игра, реализованная на чистом JavaScript с поддержкой дамок и пошагового режима.

## О проекте
Веб-приложение для игры в русские или международные шашки.  
На основе файловой структуры и коммитов реализован следующий функционал:
- Игровая доска 8×8 с классической расстановкой.
- Возможность хода как обычными шашками, так и дамками.
- Превращение шашки в дамку при достижении последней горизонтали (изображение `crown.png`).
- Пошаговый режим для двух игроков на одном устройстве.
- Подсветка доступных ходов.

## Технологии
- **JavaScript (ES6+)** — игровая логика, обработка событий, отрисовка ходов.
- **HTML5** — структура доски и элементов интерфейса.
- **CSS3** — стилизация клеток, фигур, анимации.
